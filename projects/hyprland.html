Hyprland - My First Rice


The first thing I did was to install Hyprland, a dynamic tiling Wayland compositor.


Note: You can always choose install Hyprland with pacman. Just RTFW wiki .


Install git: 

 sudo pacman -S git

Install yay: 

 git clone https://aur.archlinux.org/yay.git

Go to the yay directory: 

 cd yay

Make sure to install the base-devel: 

 sudo pacman -S --needed base-devel

Install yay: 

 makepkg -si

Now, install Hyprland: 

 yay -S hyprland-git

Anyway, make sure to install kitty (terminal): 

 sudo pacman -S kitty

Now, run Hyprland by typing: 

 Hyprland

Boom, we're good to go.
