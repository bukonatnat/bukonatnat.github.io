Arch Linux - installing, booting and setting it up.


How the hell did I install it?

• I already have a USB bootable with Arch installed, and booted into the live env.

connect it to the wifi:

iwctl

to see device list:

iwctl device-list

I normally use this command:

iwctl station wlan0 connect "your SSID"

After that, you will be ask to enter the passphrase (eg):

your password

Wait for a few seconds, and exit by typing:

exit

Now, partioning the disk. I normally have 2 disks so I run:

cfdisk /dev/nvme0n1

Delete the partitions if you have any, and create a new one. I normally create a 1G EFI partition, a 16GB swap partition, and the rest is for the root partition.

1G for EFI
16G for SWAP
the rest is for ROOT

EFI - for the boot to load.
SWAP - for the RAM, if you have low RAM, it will use this partition as a virtual memory.
ROOT - for the system files, programs, and everything else.

Now, make it ext4 by typing:

mkfs.ext4 /dev/nvme0n1p3 (root_partition)

Second, make the swap:

mkswap /dev/nvme0n1p2 (swap_partition)
